# OmniAuth AliExpress

Documentation: http://gw.api.alibaba.com/dev/doc/sys_auth.htm?ns=aliexpress.open

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-aliexpress'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-aliexpress
